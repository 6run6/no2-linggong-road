## How to build

1. 安装Qt。我现在用的版本是Qt 5.14。
2. 在QtCreator中点击Run/Debug。第一次会弹出错误对话框，别紧张，这是正常的。
3. 将项目文件夹下的data目录拷贝到可执行程序的目录下。注意，如果是macOS，需要拷贝到`{构建目录}/lgl.app/Contents/MacOS/`下。
4. 再次点击Run/Debug。现在应该可以运行了。
